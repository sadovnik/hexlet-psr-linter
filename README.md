A traineeship project for Hexlet.io

## Installation
You can install the package globally:
```
composer global require "sadovnik/hexlet-psr-linter"
```
or locally:
```
composer require "sadovnik/hexlet-psr-linter"
```

## Cli usage
`psr-linter path/to/your.php`
